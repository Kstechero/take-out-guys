# 订单取消与退款规则

## 用户主动取消

用户只能取消属于自己的待付款或待接单订单。订单进入已接单、派送中、已完成或已取消后，用户端不能再按普通取消流程处理；如有特殊情况，应联系人工客服。

取消未支付订单时，系统将订单标记为已取消。取消已支付但仍处于待接单状态的订单时，系统会发起退款，并把支付状态更新为退款。具体退款是否到账、订单当前支付状态和取消原因属于实时交易信息，必须查询订单 Tool 或支付渠道结果，不能仅凭知识文档承诺到账时间。

## 商家拒单或取消

商家只能拒绝待接单订单。商家可以取消尚未完成且尚未取消的订单。对于已支付订单，拒单或商家取消时会先执行退款流程，再把订单标记为已取消。

无论用户取消、商家拒单还是商家取消，订单已经核销的优惠券都会按订单恢复规则返还。优惠券返还后的实时可用状态仍应通过优惠券 Tool 查询。

## 自动取消

待付款订单超过 15 分钟仍未支付会自动取消，并恢复该订单占用的优惠券。网络延迟或页面状态未刷新时，以订单 Tool 查询到的服务端状态为准。

## 回答边界

本规则不承诺支付渠道的退款到账时长，也不包含人工赔付、运费补偿或线下退款政策。知识库没有依据时，应明确说明未找到确认规则，并建议联系人工客服，不得自行生成赔付金额或时限。
