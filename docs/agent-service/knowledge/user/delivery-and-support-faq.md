# 配送与客服常见问题

## 配送时间

展示预计送达时间时，优先使用订单实时状态和配送 Tool。知识库只能解释状态含义，不能替代实时查询。高峰、恶劣天气、门店备餐和配送员调度都可能导致预计时间变化。

## 订单状态

| 状态 | 含义 | 用户可做什么 |
| --- | --- | --- |
| `PENDING_PAYMENT` | 等待支付确认 | 检查支付状态，不重复提交订单 |
| `CONFIRMED` | 门店已接单 | 等待备餐，可查询预计送达 |
| `PREPARING` | 厨房备餐中 | 通常不能直接修改菜品 |
| `READY_FOR_PICKUP` | 等待配送/取餐 | 可查询配送信息 |
| `DELIVERING` | 配送中 | 联系客服处理地址或异常 |
| `COMPLETED` | 已完成 | 可评价或发起售后 |
| `CANCELLED` | 已取消 | 查询取消原因和退款状态 |

## 客服升级

以下情况不要仅依赖模板回复，应建议转人工：重复扣款、疑似食物过敏反应、退款金额争议、配送地址涉及安全风险、订单状态长时间未变化或用户无法确认身份。
