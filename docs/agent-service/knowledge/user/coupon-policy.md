# 优惠券领取与使用规则

## 领取条件

用户只能领取当前处于启用状态、位于领取有效期内且仍有剩余数量的优惠券。每张优惠券可以设置每人限领数量；用户达到限领上限后不能继续领取。并发领取时以服务端最终扣减库存的结果为准。

优惠券是否可领取、剩余数量、有效期和个人领取次数都可能实时变化，必须通过优惠券 Tool 查询，知识文档不能保证某张券当前仍可领取。

## 使用条件与金额计算

下单时只能使用当前用户已经领取、尚未使用、处于启用状态且仍在有效期内的优惠券。订单原始金额达到优惠券使用门槛后才可核销。优惠金额最多抵扣到订单原始金额，不会使最终应付金额小于 0 元。

当前后端一次订单只接收一个优惠券标识，不支持把多张优惠券叠加到同一订单。某一订单可使用哪些优惠券，必须以订单金额调用可用优惠券 Tool 计算，不能只根据券名称或描述推断。

## 状态与返还

用户优惠券状态分为未使用、已使用和已过期。优惠券在订单创建过程中完成核销；如果订单随后因用户取消、商家拒单、商家取消或支付超时自动取消，系统会按订单恢复对应优惠券。

恢复后是否仍在有效期内、是否可以再次用于新订单，需要重新查询优惠券 Tool。知识库不承诺过期券延期，也不承诺额外补发优惠券。

## 无依据场景

本项目没有统一的优惠券兑换码、跨店通用、现金提现或与其他活动叠加规则。遇到这些问题时，应说明当前知识库没有确认依据，不得套用其他平台规则。
